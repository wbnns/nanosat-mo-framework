NMF Connector in C

by NMF

Posted on May 16, 2017

The objective of this task is to develop a connector to the NMF in C programming language. An App developer is expected to use this component to be able to develop apps in C that are able to acquire data from the Platform services and/or other services.

The activity consists on the following sub-tasks:

- Familiarization with MO

- Familiarization with the MO C implementation available online: https://github.com/ccsdsmo/malc

- Automatic generation of the stubs and skeletons for the services available on the NMF.

- Implementation of the connections logic to the service consumers

- Wrap-up into a single component that can be "ready to be used" by other developers.

Login service implementation (Finished)

by NMF

Posted on May 16, 2017

The objective of this task is to implement a Login service using an already existing security framework such as Apache Shiro. The CCSDS has defined an MO interface in the CCSDS Common services set for the authentication and autorization of consumers to a provider.

The activity consists on the following sub-tasks:

- Familiarization with MO

- Familiarization with the NMF implementation

- Implementation of the Login service

The source code for ths project is available here:

https://github.com/andreeap/CCSDS-MO-Login-Service-Implementation
